# SafeHome Mobile Prototype

SafeHome 제품의 모바일 뷰 프로토타입입니다.

## 주요 기능

- **Dashboard**: 홈 현황 및 주요 정보 대시보드
- **Emergency**: 긴급 상황 관리 및 알림
- **Device**: 연결된 스마트 홈 기기 관리
- **Preference**: 사용자 설정 및 환경설정

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 기술 스택

- React 18
- Vite
- CSS3 (모바일 최적화)
